# React Solidity Boilerplate dApp

## How to start

```
npm i
npm start
```

## Deploy live

```
./build.sh [folder] [port] install
```

## Test

```
npm start
npm run cypress:open
npm run cypress:run
```

## Analyze bundle

```
npm run analyze
```

* [React](https://github.com/facebook/react)
